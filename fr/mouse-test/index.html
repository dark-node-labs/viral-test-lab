FAQ Test Souris

Que peut diagnostiquer un test souris ?

Il peut aider à repérer les problèmes de boutons, double-clic, molette et mouvement du pointeur.

Pourquoi le clic droit est-il spécial ?

Le navigateur ouvre normalement un menu contextuel, donc la zone de test le bloque pour détecter le bouton.

Faut-il comparer sur le même appareil ?

Oui. Le navigateur, le système, le pilote et la connexion peuvent modifier le résultat.
