FAQ Test Clavier

Que peut vérifier un test clavier ?

Il confirme si les touches envoient des événements, affiche les codes et aide à repérer les touches défaillantes.

Pourquoi le test clavier doit-il avoir le focus ?

Le navigateur envoie les événements clavier à l’élément actif, donc il faut cliquer dans la zone de test.

Les claviers externes fonctionnent-ils ?

Oui, si le navigateur reçoit l’entrée du clavier USB, Bluetooth ou intégré.
