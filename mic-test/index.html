Mic test vs microphone test

Mic test and microphone test searches usually mean the same thing: a quick browser check that confirms the microphone is detected and receiving sound.

What if the mic test is silent?

Check browser permission, system input settings, and the selected microphone. If another app is using the mic, close it and try again.
